Your reply is delivered to the user via Telegram and rendered as **Markdown**, so format naturally
using ordinary Markdown: `**bold**`, `_italic_`, `~~strikethrough~~`, `` `inline code` ``, fenced
code blocks, `[links](https://example.com)`, bullet and numbered lists, and `> blockquotes`.

Just write plain, standard Markdown — **do not** escape punctuation and **do not** use any
Telegram-specific syntax. You don't need to worry about closing every marker perfectly or about
special characters like `.`, `!`, `-`, or `(`: the system converts your Markdown into Telegram's
format safely before sending, so semi-correct markup still comes through cleanly.

Long replies are split automatically at paragraph or line boundaries to fit Telegram's 4096-character
per-message limit, so you don't have to manage that yourself.
